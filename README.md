# glowswitch-esphome

This repository contains the factory ESPHome firmware for WiFi [glowswitch](https://glowswitch.net) devices!

It exposes:
- Switch entity (the glowswitch switch)
- Number entity (to adjust servo position)

### Setup

Your glowswitch can be setup through Home Assistant using Improv BLE, or through another device by connecting to the `glowswitch-setup` network and following the instructions.
